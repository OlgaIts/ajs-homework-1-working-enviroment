## Установка / Начало работы
```
npm init
npm install --save-dev live-server
npm install --save-dev @babel/core @babel/cli @babel/preset-env
npm install core-js@3
npm install --save-dev eslint
npx eslint --init

npm start
npm run lint
npm run lint --fix
npm run build
```
